# Explo
## Dependencies
Google Maps:  
```
npm install react-native-maps --save  
cd ios  
pod install  
```
  
## Running on iOS Simulator
In Explo main directory, run:  
```
npx react-native run-ios
```

## Running on Expo App
In Explo main directory, run:  
```
npx expo start
```
You may need to install some dependencies, such as:
```
yarn install expo
```
